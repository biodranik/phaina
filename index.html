PHaina — генератор мультиязычных статических сайтов на PHP

Быстро и удобно сделать себе сайт и бесплатно захостить его на GitHub.

В чём фишка?

PHaina — генератор статических сайтов с кастомными страницами, который умеет:

  • Добавлять и кастомизировать страницы на чистом HTML или с примесью PHP
  • Удобно разрабатывать сайт и отлаживаться локально
  • Мультиязычность
  • Генерировать статический html и ресурсы, готовые для любого хостинга
  • Бесплатно хоститься на GitHub
  • …И многое другое
